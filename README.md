# 🚀 EspOneClickInstaller

Instalador automático para configuração completa de placas **ESP32 / ESP8266**.

---

## ⚙️ O que ele faz

* Instala e configura a **Arduino IDE**
* Adiciona suporte para **ESP32 / ESP8266**
* Instala drivers USB:

  * CP210x
  * CH340 / CH341
* Deixa tudo pronto para uso em poucos minutos

---

## ▶️ Como usar

1. Baixe o arquivo
2. Execute como **Administrador**
3. Aguarde a instalação

---

## 💻 Requisitos

* Windows
* Internet
* Permissão de administrador

---

## ⭐ Sobre

Projeto criado para simplificar a configuração de ambientes ESP, evitando erros comuns de instalação manual.
